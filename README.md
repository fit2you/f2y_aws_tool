# F2yAwsTool

AWS OpsWorks Stack Deployment CLI


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'f2y_aws_tool'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install f2y_aws_tool

## Usage

    $ ./bin/f2y-aws-tool help

## Test

    $ bundle exec rspec
